<h1 align="center">Quark Design</h1>

<div align="center">
  
  Next-gen frontend component library, it can be used in any framework or no framework.
  
</div>

哈啰集团面向C端UI组件库，支撑哈啰几乎所有C端H5项目，包括交易，支付，两轮，商城等。这是一个设计轻盈优雅的企业级UI组件库，可以满足所有日常使用的基础组件需求。
